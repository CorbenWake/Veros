Здесь буду записывать все команды и нюансы работы с ГИТ:

git clone path-to-repository git status - статус текущего репозитория git add - добавить новый файл под контроль ГИТ( --aa -добавить все файлы каталога) git commit -am "added new files index.html and git html" git push - отправить изменения на сервер(после коммита) git pull - забрать с сервера
